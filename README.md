# 穴位测验大师 v2.4.0 🎯

一个专业的中医针灸学习应用，支持穴位抽取练习、AI智能测评功能和无障碍学习体验。

## 🆕 v2.4.0 更新亮点

### ♿ 无障碍功能增强
- ✨ **全新无障碍动作系统**：经络和穴位选择器支持TalkBack无障碍动作
- 🎯 **精确导航控制**：在穴位学习页面，使用"下一个经络"、"上一个经络"、"下一个穴位"、"上一个穴位"动作
- 📱 **TalkBack优化**：移除旧的手势滑动功能，改为标准无障碍动作，更符合用户习惯
- 🔊 **语音提示完善**：改进内容描述，提供更清晰的语音反馈

### 🛠️ 技术升级
- 📱 **API版本提升**：最低支持Android 7.0 (API 24)，更好兼容现代无障碍特性
- 🔧 **AndroidX支持**：集成AndroidX Core库，提供最新的无障碍API

## 📱 功能特色

### 🎲 穴位抽取练习
- 支持16个经络题库的随机抽取
- 包含手太阴肺经、手阳明大肠经、足阳明胃经等完整经络系统
- 支持盲医考必背穴位专项练习
- 实时统计答题正确率

### 🤖 AI智能测评
- 支持多个主流AI平台
  - DeepSeek（深度求索）
  - OpenAI（GPT系列）
  - Google Gemini
  - Anthropic Claude
  - 阿里通义千问
  - 百川智能
  - 智谱ChatGLM
  - 自定义平台
- 智能出题：定位、主治、归经等多维度测试
- 智能判卷：多维度评分系统，准确评估答题质量
- 详细反馈：提供改进建议和标准答案对比

### 📚 穴位学习模块
- **经络选择**：从16个经络中选择要学习的经络
- **穴位浏览**：查看每个穴位的详细信息
- **无障碍导航**：支持TalkBack用户快速切换经络和穴位

### ♿ 无障碍设计
- **TalkBack全支持**：完整的屏幕阅读器支持
- **无障碍动作**：自定义无障碍动作，便于单手操作
- **语音描述**：清晰的内容描述和状态反馈
- **键盘导航**：支持外接键盘导航

### ⚙️ 便捷设置
- 一键选择AI平台，自动配置API地址
- 支持自定义API配置
- 连接测试功能确保设置正确

## 🔧 无障碍功能使用指南

### TalkBack用户
1. **启用TalkBack**：设置 → 辅助功能 → TalkBack
2. **使用无障碍动作**：
   - 在穴位学习页面，聚焦到经络选择器
   - 双指上滑打开TalkBack菜单
   - 选择"动作"，找到"下一个经络"或"上一个经络"
   - 同样适用于穴位选择器的"下一个穴位"和"上一个穴位"

### 快捷操作
- **经络切换**：在经络选择器上使用无障碍动作快速切换
- **穴位浏览**：在穴位选择器上使用无障碍动作浏览不同穴位
- **语音反馈**：每次切换都会有清晰的语音提示

## 🛠️ 技术栈

- **开发框架**：.NET 9 for Android
- **最低版本**：Android 7.0 (API 24)
